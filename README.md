# GodlyForge

**The ultimate web studio for creators.**

Turn any idea into a complete, attention-grabbing, Gumroad-top-tier digital product in minutes.

Powered by the embedded **GODLY multi-stage creation engine** that consistently produces Notion templates, AI prompt packs, ebooks, planners, micro-courses, and design kits with professional polish, sales-page copy, mockups, pricing, and one-click exports.

## Tech Stack
- Next.js 15 (App Router) + React 19
- TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- Zustand
- Lucide React

## Features
- Dashboard with trending niches
- Smart Idea Generator
- Product Builder (6 types)
- GODLY Generator Service
- Live Sales Page Preview
- Product Library & Export (PDF/Markdown/JSON)
- Fully responsive, dark-mode-first, accessible

## Getting Started
```bash
npm install
npm run dev
```
